# Archive

## Archive policy

- Files here may describe completed cleanup work, earlier architecture assessments, or superseded recommendations.
- Prefer active top-level project docs for the current transmitter architecture and workflow.
- Keep archive references explicit so historical material is clearly separated from active guidance.
